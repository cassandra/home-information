{% load units %} {% comment %} Continuous slider widget for controllers whose value is in ``EntityState.units`` and must be translated to the user's preferred display unit at the render boundary. Translation goes through the ``to_display`` filter (backed by ``ConsoleConverterHelper``); the HTML/JS layer never sees the canonical value. For unit-agnostic / fixed-suffix sliders (hue, saturation, percent, etc.), use ``continuous_slider.html`` instead.
