Icon Editing
  • Click Select icon
  • Drag Move icon

  • ESC Deselect

Path Editing
  • Click Select path to edit vertices
  • Drag Point Move single vertex
  • CTRL + Drag Move entire path
  • I / INSERT Insert vertex
  • X / DEL Delete vertex
  • A / + Add new path segment
  • CTRL + X Delete entire path
  • ESC Finish path editing
