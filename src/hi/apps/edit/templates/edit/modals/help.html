Moving

  • Left mouse click and drag
  • One finger drag

Scaling

  • Mouse wheel
  • "+" key (zoom in)
  • "-" key (zoom out)
  • "s" key and drag
  • Two finger pinch/spread
  • "ESC" key to end

Rotating

  • "r" key and drag
  • "r" key and mouse wheel
  • "r" then "+" key (rotate right)
  • "r" then "-" key (rotate left)
  • Two finger rotate
  • "ESC" key to end

Path Editing

  • Click to select area
  • Click and drag points to change area's shape
  • Click point of edge to select
  • "i" or "INSERT" key to insert path node
  • "x", "DEL" or "BACKSPACE" key to delete path node
  • "a" or "+" key - add new path node or path segment
  • "ESC" key to end path editing

Notes

  • Background edits of view only saved when the "View" editing pane is showing.
