Scaling
  • Mouse Wheel Scroll to zoom
  • + / - Zoom in/out

  • Pinch Two finger gesture
  • ESC End scale mode

Rotating
  • R + Drag Rotate mode
  • R + Wheel Rotate with mouse
  • R + +/- Rotate right/left
  • Two Finger Touch rotate
  • ESC End rotate mode

Path Editing
  • Click Select area
  • Drag Points Change shape
  • I/INSERT Insert path node
  • X/DEL Delete path node
  • A/+ Add new segment
  • ESC End path editing
